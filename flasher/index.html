Put the board into download mode

The web flasher talks to the ESP32-C3 bootloader over USB serial. If the installer can’t connect, you likely need to enter Download Mode.
Download Mode (BOOT + RESET)
  1. Plug the board into USB.
  2. Hold BOOT.
  3. Tap RESET (or unplug/plug while holding BOOT).
  4. Release BOOT.
  5. Click Install below and pick the board’s serial port.
Warning: Flashing overwrites firmware. Pick the correct image for your board. If you’re unsure, start with the Factory build.
After flashing, you can open the board UI at http://cyberman-hw-ctf-xxxxxx.local (mDNS) or use the IP shown on your router / serial console.
